# Encryption Examples

This section contains examples demonstrating how to encrypt values using FHEVM. Encryption is the process of converting plaintext values into encrypted format that can be used in fully homomorphic operations.

## Overview

The encryption examples show how to:

- Convert plaintext values to encrypted format
- Handle single and multiple encrypted values
- Set up proper permissions for encrypted data
- Work with external encrypted inputs from clients
